# Nego
1. 목표  
실제 중고거래 서비스의 사용자 흐름을 분석하고, 회원 가입부터 게시글 등록, 조회, 거래로 이어지는 전체 과정을 구조적으로 이해하는 것을 목표로 합니다. 이를 바탕으로 사용자가 서비스를 이용하는 흐름에 맞춰 게시글 등록·조회·거래 기능을 구현하고, 각 기능이 유기적으로 연결되도록 설계 히였습니다.
이 과정을 통해 단순한 기능 구현을 넘어, 웹 서비스가 어떤 구조로 동작하는지와 백엔드에서 데이터와 비즈니스 로직이 어떻게 관리되는지를 경험하며 전반적인 백엔드 개발 역량을 강화하고자 했습니다.
